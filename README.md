# CraftKit

Free browser-based tools for makers. No signup, no BS.

## Quick start

```bash
npm install
npm run dev
```

No API keys needed — everything runs client-side.

## Tools

- **ShotCraft** — Screenshot beautifier. Drop an image, pick a background, export a polished shot.
- **InvoiceCraft** — Invoice generator. Fill in details, download a clean PDF.
- **PolicyCraft** — Privacy policy generator. Answer a few questions, get a ready-to-use policy.
- **QRCraft** — Styled QR codes. Enter a URL, customize colors and style, download.

## Stack

Next.js · Tailwind CSS · Lucide Icons
